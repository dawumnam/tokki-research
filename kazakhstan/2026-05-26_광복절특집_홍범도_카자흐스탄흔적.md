# 카자흐스탄 광복절 특집 — 홍범도의 카자흐스탄 흔적 (에피소드 5 + 현존 장소)

## 1. 개요

봉오동·청산리 전투의 영웅 홍범도(洪範圖, 1868~1943)는 <a id="cite-1-1"></a>[[1]](#ref-1-1)1937년 스탈린의 고려인 강제이주로 연해주에서 카자흐스탄 크즐오르다(Kyzylorda)로 옮겨져 그곳에서 생을 마쳤다. 이 조사는 **크즐오르다 시절의 에피소드 5개**(강제이주·정착 / 고려극장 수위 근무 / 태장춘 희곡 「홍범도」 관람 / 「홍범도 일지」 구술 / 1943년 사망·매장)와, <a id="cite-1-2"></a>[[2]](#ref-1-2)그 에피소드와 연결되며 **2026년 현재 현지에서 촬영 가능한 장소**(홍범도 장군 기념공원·홍범도 거리·국립 고려극장)를 정리한 것이다. 유해는 <a id="cite-1-3"></a>[[3]](#ref-1-3)2021년 8월 대한민국으로 봉환되어 국립대전현충원에 안장됐으나, 옛 묘역 자리는 기념공원으로 남아 있다.

### 2.1 에피소드① — 1937년 강제이주와 크즐오르다 정착

**핵심 주장**: <a id="cite-2-1-1"></a>[[1]](#ref-2-1-1)홍범도는 1937년 스탈린의 고려인 강제이주로 연해주에서 카자흐스탄으로 이송돼 <a id="cite-2-1-2"></a>[[2]](#ref-2-1-2)1938년 4월 크즐오르다에 정착했다.

**근거 및 설명**:

- 1937년 9월 단행된 고려인 강제이주로 연해주 한인 약 17만 명이 열차에 실려 중앙아시아로 옮겨졌다. 기념사업회 일대기는 "원문: <a id="cite-2-1-3"></a>[[3]](#ref-2-1-3)1937년 9월, 스탈린은 근대 세계사에서 찾아볼 수 없는 '민족 강제 집단 이주'를 단행하였다"라고 적는다.
- 홍범도는 이주 지역 중 가장 먼 곳인 카자흐스탄 얀꾸르간(야니쿠르간) 시르다리야 강 건너편 사나리크 셀소비에트(카잘린스크 구역)로 이송됐다. 뉴시스는 "<a id="cite-2-1-4"></a>[[1]](#ref-2-1-4)홍 장군은 기차에 실려 중앙아시아 카자흐스탄 얀꾸르간 시르다리아 강 건너편의 사나리크 셀소비에트(카잘린스크 구역)로 이주했다"고 보도했다.
- 이듬해 <a id="cite-2-1-5"></a>[[1]](#ref-2-1-5)1938년 4월 초 크즐오르다 도시구역 크라스늬 고로독("붉은 구석") 60번지로 이사했다.
- 정착 초기 홍범도는 <a id="cite-2-1-6"></a>[[3]](#ref-2-1-6)독립군 부대원 50여 명과 농업조합(집단농장)을 만들어 소련 정부로부터 개간 허가를 받아 목화·귀리·수수를 경작했다.

### 2.2 에피소드② — 크즐오르다 고려극장 수위(수직원) 근무

**핵심 주장**: <a id="cite-2-2-1"></a>[[1]](#ref-2-2-1)홍범도는 크즐오르다 정착 후 고려극장의 수직원(수위장)으로 일하며 무대기구를 지켰고, <a id="cite-2-2-2"></a>[[2]](#ref-2-2-2)극장 수위 등으로 일하다가 말년을 보냈다.

**근거 및 설명**:

- 정착 직후 홍범도는 병원 경비로 어렵게 일했다. 뉴시스: "<a id="cite-2-2-3"></a>[[1]](#ref-2-2-3)그는 병원의 경비로 어렵게 일했다."
- 사정을 알게 된 고려극장 측이 월 50루블을 받는 수직원(수위장) 자리를 마련해 줬다. 뉴시스: "<a id="cite-2-2-4"></a>[[1]](#ref-2-2-4)홍 장군 사정을 알게 된 고려극장 관계자들이 홍범도에게 매달 50루블을 받는 수직원(수위장) 자리를 만들어줬다."
- 업무는 전용 건물이 없던 극장이 빈 창고에 모아둔 무대기구를 지키는 일이었다. 뉴시스: "<a id="cite-2-2-5"></a>[[1]](#ref-2-2-5)홍 장군은 전용 건물이 없는 극장인 고려극장의 빈 창고에 모아둔 무대기구를 지키는 임무를 맡았다."
- 한국민족문화대백과사전은 말년 행적을 "<a id="cite-2-2-6"></a>[[2]](#ref-2-2-6)이곳에서 극장 수위 등으로 일하다가 1943년 76세로 사망하였다"고 기술한다.

### 2.4 에피소드④ — 「홍범도 일지(일기)」 구술

**핵심 주장**: <a id="cite-2-4-1"></a>[[1]](#ref-2-4-1)홍범도는 말년에 자신의 일대기를 구술·기록으로 남겼고, 그 「홍범도 일지(일기)」는 오늘날까지 전한다.

### 2.5 에피소드⑤ — 1943년 사망과 크즐오르다 매장

**핵심 주장**: <a id="cite-2-5-1"></a>[[1]](#ref-2-5-1)홍범도는 1943년 10월 25일 크즐오르다에서 별세해 현지에 매장됐고, <a id="cite-2-5-2"></a>[[2]](#ref-2-5-2)이후 묘비 건립·이장을 거쳐 묘역이 조성됐다.

**근거 및 설명**:

- 한국민족문화대백과사전: "<a id="cite-2-5-3"></a>[[1]](#ref-2-5-3)1937년 스탈린의 한인강제이주정책에 의하여 카자흐스탄의 크질오르다로 강제이주되어 이곳에서 극장 수위 등으로 일하다가 1943년 76세로 사망하였다."
- 묘역 변천: <a id="cite-2-5-4"></a>[[2]](#ref-2-5-4)1951년 고려인들이 모금해 묘비를 세웠고, 1982년 4월 25일 중앙공동묘지 현재 위치로 이장, 1984년 10월 25일 반신 청동 흉상과 추모비가 건립됐다.
- 1982~83년 이장 과정에서 사라졌던 1951년 묘비가 <a id="cite-2-5-5"></a>[[3]](#ref-2-5-5)2024년 10월 9일 크즐오르다 현지에서 40여 년 만에 발견됐는데, 앞면에 한글로 "저명한 조선빨찌산대장 홍범도 묘"라고 새겨져 있었다.
