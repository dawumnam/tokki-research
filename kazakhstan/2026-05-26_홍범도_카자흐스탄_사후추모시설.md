# 홍범도 장군 사후 추모 시설 (카자흐스탄)

## 1. 개요

이 조사는 홍범도(1868~1943, 봉오동·청산리 전투의 독립군 사령관, 카자흐스탄 크즐오르다에서 서거)의 **사후에** 카자흐스탄에 조성된 추모·기념 시설을 다룬다. 그 결과 (1) 크즐오르다 옛 묘역·반신청동상·추모비, (2) 크즐오르다 홍범도 기념공원(2023년 개원)·홍범도·계봉우 전시관, (3) 크즐오르다 홍범도 거리(현판 2023년 부착), (4) 1951년 최초 묘비(현재 한국으로 일부 봉환), (5) 알마티 고려극장 내 홍범도 전시·초상화 등 **5건 이상의 추모 시설**을 확인했다 <a id="cite-1-1"></a>[[1]](#ref-1-1)<a id="cite-1-2"></a>[[2]](#ref-1-2)<a id="cite-1-3"></a>[[3]](#ref-1-3). 홍범도 유해는 2021년 8월 한국으로 봉환돼 대전현충원에 안장됐으나, 옛 묘역의 흉상·석물·기념공원은 그대로 현지에 남아 촬영 가능하다 <a id="cite-1-4"></a>[[4]](#ref-1-4).

## 2. 조사 결과

### 2.1 크즐오르다 옛 묘역 — 반신청동상(흉상)·추모비

**핵심 주장**: 크즐오르다 중앙공동묘지의 홍범도 옛 묘역에는 고려인들이 **1984년 10월 25일 건립한 반신청동상(흉상)과 추모비**가 있으며, 2021년 유해 봉환 이후에도 흉상과 석물이 현지에 그대로 남아 있다 <a id="cite-2-1-1"></a>[[1]](#ref-2-1-1)<a id="cite-2-1-4"></a>[[4]](#ref-2-1-4).

**근거 및 설명**:

- 묘역 연혁: "1982년 4월 25일, 중앙공동묘지의 뒤편에서 현재의 넓고 좋은 위치로 이장"했고, "1984년 10월 25일, 홍범도 장군의 반신청동상과 추모비 건립" <a id="cite-2-1-1"></a>[[1]](#ref-2-1-1).
- "1996년 5월 20일, 한국 정부와 기업 제공 기금으로 홍범도 장군 흉상 주변에 3개의 기념비를 추가 건립, 묘역 조성·단장" <a id="cite-2-1-1"></a>[[1]](#ref-2-1-1).
- 유해 봉환 후 현장 상태: "유해를 봉환했음에도 흉상과 석물들이 그대로 남아있는 현장에는 작은 기념관을 짓는 공사"가 진행되었다 <a id="cite-2-1-4"></a>[[4]](#ref-2-1-4).

### 2.2 크즐오르다 홍범도 기념공원 (2023년 개원) — 홍범도·계봉우 전시관

**핵심 주장**: 국가보훈부가 홍범도 옛 묘역 일대를 **2023년 11월 3일 개원한 기념공원**으로, 부지 1,067㎡(323평)에 봉오동 전투 지세를 형상화한 참배 공간·홍범도/계봉우 전시관·휴게공간으로 구성됐다 <a id="cite-2-2-2"></a>[[2]](#ref-2-2-2)<a id="cite-2-2-3"></a>[[3]](#ref-2-2-3).

- "카자흐스탄 크즐오르다시 이반주르바거리에 위치한 홍범도 장군 옛 묘역"이며 "부지면적 1067㎡(323평)" <a id="cite-2-2-2"></a>[[2]](#ref-2-2-2).
- 구성: "봉오동 전투의 지세를 형상화한 참배 공간을 비롯해 홍범도 장군과 계봉우 지사의 삶과 독립운동 공적을 전시하는 전시관, 조경과 차양 쉼터가 있는 휴게공간" <a id="cite-2-2-2"></a>[[2]](#ref-2-2-2).

### 2.4 1951년 최초 묘비 (고려인 건립, 현재 일부 한국 봉환)

**핵심 주장**: 고려인들이 십시일반 모금해 세운 **최초의 묘비**(철제, "저명한 조선빨찌산대장 홍범도 묘" 한글 표기)가 1982년 이장 때 보관됐다가 2024년 발견됐고, 그 일부가 2024년 한국으로 전달됐다 <a id="cite-2-4-8"></a>[[8]](#ref-2-4-8)<a id="cite-2-4-9"></a>[[9]](#ref-2-4-9).

- 최초 묘비 제원: "고려인들이 십시일반 돈을 모아 세운 최초의 묘비"로, 철로 제작, 크기 세로 44㎝·가로 30㎝, 앞면에 "저명한 조선빨찌산대장 홍범도 묘"라고 한글 표기 <a id="cite-2-4-8"></a>[[8]](#ref-2-4-8).

- 보관 경위: 1982년 이장으로 흉상·추모비가 새로 서며 옛 묘비가 갈 곳이 없자, '카자흐스탄 홍범도 기념사업회' 회장 김 레프 니콜라예비치(김례프)가 묘비를 받아 자택에 보관 <a id="cite-2-4-8"></a>[[8]](#ref-2-4-8).
