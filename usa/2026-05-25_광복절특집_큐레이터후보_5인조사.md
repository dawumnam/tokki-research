# 광복절 특집 큐레이터 후보 5인 상세 조사

## 1. 개요

미국 광복절 특집편 큐레이터 후보 5명(임경석·윤해동·오제연·이기훈·김학재)에 대해 생년, 소속·직위, 전공, 유튜브 레퍼런스, 영어 능력 증거, 교포 여부를 중심으로 조사했다. <a id="cite-1-1"></a>[[1]](#ref-1-1) 조사 결과: 임경석(1958년생)은 출생연도 조건 미충족 확실, 현재 명예교수. 윤해동은 생년 미확인(추정 1959~1962년대 초반, 조건 미충족 가능성 높음), 소속 변경 이력 있음. 오제연은 생년 미공개이나 영문 논문 확인. 이기훈은 2005년 박사 취득 기준 추정 1975년 전후 출생 가능, 유튜브 강의 미확인. 김학재는 2013년 박사 취득, 출생연도 미공개, 영문 저서 확인. <a id="cite-1-2"></a>[[2]](#ref-1-2)

### 2.1 임경석 (성균관대 사학과 명예교수, 사회주의 독립운동사)

**핵심 주장**: **출생연도 1958년으로 확인되어 조건(1963~1997년) 미충족. 현재 명예교수(재직 중 아님). 후보 제외 사유 명확.** <a id="cite-2-1-1"></a>[[1]](#ref-2-1-1)

**근거 및 설명**:

- **생년**: 1958년, 전남 여수 출생. <a id="cite-2-1-2"></a>[[1]](#ref-2-1-1) 출생연도 조건(실질 1963~1997년) 미충족 — 허용 하한(±1~2살 적용 시 1963년) 대비 5년 초과.
- **현재 직위**: 성균관대학교 사학과 **명예교수**. 교원정보 시스템에서 직위가 "명예교수"로 표기됨. <a id="cite-2-1-3"></a>[[2]](#ref-2-1-3) 현역 교수 아님.
- **학력**: 성균관대학교 사학과 학사·대학원 졸업(박사). 박사 논문 제목: 〈고려공산당 연구〉. <a id="cite-2-1-4"></a>[[1]](#ref-2-1-1)
- **전공**: 한국 근대사, 일제하 사회주의 운동사. 1982년 대학원 입학, 이후 한국 사회주의·독립운동사 연구에 집중. 1994~1995년 러시아 코민테른 문서보관소에서 18개월 아카이브 연구. <a id="cite-2-1-5"></a>[[3]](#ref-2-1-3)
- **주요 저서**: 『한국 사회주의의 기원』(역사비평사, 2003), 『독립운동 열전』 1·2(푸른역사, 2022), 칼럼 연재 「임경석의 역사극장」(한겨레21). <a id="cite-2-1-6"></a>[[1]](#ref-2-1-1)

### 2.2 윤해동 (성균관대 동아시아학술원 HK교수, 식민지 근대성)

**핵심 주장**: 출생연도 미공개. 2010년 기사에서 나이를 역산하면 **추정 1959년 전후 출생**으로, 조건 하한(1963년) 미달 가능성이 높다. 현재 소속은 성균관대 동아시아학술원. 영문 출판물 확인(Palgrave 2013). <a id="cite-2-2-1"></a>[[1]](#ref-2-2-1)

**근거 및 설명**:

- **생년 추정 근거**: 최초 검색 시 AI 요약본에서 "2010년 기사에서 51세로 기록" 정보가 포함됨. 정확한 원문 확인은 불가하여 미검증 상태. 추정 시 1959년생으로 조건 미충족 가능성 높음. <a id="cite-2-2-2"></a>[[2]](#ref-2-2-2)
- **소속·직위**: 성균관대학교 동아시아학술원 HK교수(한국학술진흥재단 지원 인문한국 교수직). 한양대학교 비교역사문화연구소 교수 이력 있음(전직). <a id="cite-2-2-3"></a>[[1]](#ref-2-2-1)
- **학력**: 서울대학교 국사학과 학사·대학원(박사). 와세다대학교 외국인 연구원 역임. <a id="cite-2-2-4"></a>[[1]](#ref-2-2-1)
- **전공**: 식민지 근대성(colonial modernity) 연구. 한국 근대 농촌·임야 공유지 변화, 식민지 공공성, 포스트식민주의 역사학. <a id="cite-2-2-5"></a>[[1]](#ref-2-2-1)
- **주요 저서**: 『식민지의 회색지대』(2003), 『지배와 자치』(2006), 『식민지 근대의 패러독스』(2007). <a id="cite-2-2-6"></a>[[1]](#ref-2-2-1)
- **유튜브 레퍼런스**: 웹 검색 범위 내 개인 강의 영상 URL 미확인.
- **영어 능력 근거**: "Colonial Publicness as Metaphor" — Michael Kim 번역, *Mass Dictatorship and Modernity* (Palgrave Macmillan, 2013), pp.159-177에 수록. <a id="cite-2-2-7"></a>[[3]](#ref-2-2-3) 원저 논문의 영어 번역 수록으로 국제 학술 출판 실적 있음.
- **교포 여부**: 해당 없음(국내 거주·재직).

### 2.3 오제연 (성균관대 사학과 부교수, 한국현대사)
