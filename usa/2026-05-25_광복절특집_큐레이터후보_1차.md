  - **김학재** — 생년만 확인하면 전 조건 충족 (서울대 통일평화연구원 행정실 02-880-2089)
  - **류미나** — 유튜브 영상만 확인되면 전 조건 충족 (국민대 직접 연락)
- **제작진 판단 필요 조건부 후보**:
  - **정준영** — 1963년생 (허용 하한 정확히 해당), 사회학 박사·규장각 교수, 교토대 외국인연구원 경력

### 핵심 제약 사항

"한국 근현대사 전문가 + 유튜브 역사 강의 영상 + 해외 실제 수학/근무 경력 + 1963년 이후 출생 + 비교포 + 한국 거주"를 **동시에** 충족하는 학자는 극히 드뭅니다. 총 **45명+** 검토 후 현재 파이프라인이 공개 자료 기반 탐색의 실질적 한계입니다.

### 최종 단계 권고 (섭외 우선순위 순)

1. **허수·권보드래** — 즉시 섭외 접촉 시작
2. **김학재** — 서울대 통일평화연구원 행정실 (02-880-2089) 전화 → 생년 확인 → 범위 내이면 즉시 섭외
3. **류미나** — 국민대 교양대학 직접 연락 또는 와세다대 조선문화연구소 채널 탐색 → 유튜브 영상 확인 → 있으면 즉시 섭외
4. **정준영** — 제작진이 ① 1963년생 허용 여부, ② 사회학 박사 역사 관련 학위 인정 여부 결정 후 서울대 규장각 연락
